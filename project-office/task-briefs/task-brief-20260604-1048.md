## [Architect] Design

**Feature summary:** The `GET /api/v1/experience/fno/portfolio-analytics?mode=real|mock` endpoint is a new Experience-tier read-only endpoint that delivers a single, unified analytics payload covering positions, per-instrument greeks, net greeks, σ-based scenario levels, a payoff grid, stress test results, hedge quality scores, and live market data — all derived from the user's saved equity portfolio and hedge plan. This endpoint replaces the broken multi-call chain in `app-init.js` (summary + fetchPositions + injectAsmlToState) as the single source of truth for all FnO dashboard sections.

**API contract:**
| Field | Value |
|---|---|
| Method | GET |
| Path | `/api/v1/experience/fno/portfolio-analytics` |
| Query params | `mode` (string, optional, "real"\|"mock", default "real") |
| Request body | None |
| Auth required | mode=mock: no JWT; mode=real: JWT required → 401 if absent |
| HTTP status | 200 (mock always), 200 (real with portfolio), 401 (real no JWT), 404 (real valid JWT no portfolio) |
| Response shape | `mode`, `portfolio_meta` (name, total_value_eur, updated_at), `market` (per-instrument OHLCV+chg), `positions` (und, full, exp="EQUITY", type="EQ", side="Long", qty, allocation_pct, position_eur, avg, ltp, chg, pnl, currency, ann_vol_pct, region), `greeks` (und, exp, delta, gamma, theta, vega, allocation_pct, ann_vol_pct, sigma_eur, hedge_type, put_cost_eur, call_income_eur, net_theta_eur_day), `net_greeks` (delta, theta, vega), `net_delta` (per-instrument), `scenario_levels` (per-instrument: target, sl), `payoff` (portfolio+hedged: labels+data), `stress` (5 events: label, move_pct, portfolio_pnl_eur, hedged_pnl_eur), `hedge_quality` (positions array: instrument, hqs, hqs_tier, hedged, strategy, coverage_pct, note), `closed_positions` (always []), `realized_pnl` (always 0), `margin` (always {}) |

**Frontend target:**

| Section id | None — Phase 2 |
| DOM element IDs | None — Phase 2 |
| Window bindings | None — Phase 2 |

**Edge cases to handle:**
- E1: mode=mock, no JWT → 200 with MOCK_PORTFOLIO, zero DB calls
- E2: mode=real, no JWT → 401
- E3: mode=real, valid JWT, no portfolio key → 404
- E4: mode=real, valid JWT, key exists but no portfolio → 404
- E5: mode=real, portfolio exists, no hedge plan → safe defaults (delta=1.0, theta/vega/gamma=0)
- E6: instrument absent from market_data_cache → graceful degradation (fallback vol 25%, ltp=0)
- E7: portfolio with empty holdings list → 200 with empty arrays/dicts
- E8: total_value_eur is None → default to 0, position_eur=0, sigma_eur=0
- E9: invalid mode value → 422 (FastAPI Literal validation)
- E10: sigma≈0 or S/K causes math.log domain error → try/except guard, return 0.0
