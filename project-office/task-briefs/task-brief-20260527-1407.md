## [Architect] Design

**Feature summary:** Add a new FnO dashboard page that shows ASML equity portfolio performance over a user-selected date range, computes Black-Scholes covered call and protective put scenarios, and renders a payoff comparison chart across a price grid. The user can configure instrument, share count, and date range via a Portfolio Builder panel.

**API contract:**
| Field | Value |
|---|---|
| Method | POST |
| Path | `/api/v1/portfolio/equity-hedge-scenarios` |
| Query params | none |
| Request body | instrument: str, n_shares: int, start_date: str, end_date: str |
| Response shape | portfolio{instrument, n_shares, start_price, end_price, return_pct, vol_30d_pct, portfolio_value, daily[]}, hedge_scenarios{mild_bearish{strategy, strike_label, premium_per_share, total_premium_eur, max_value_eur, breakeven_price, description}, strong_bearish{strategy, strike_label, premium_per_share, total_premium_eur, floor_value_eur, breakeven_price, description}, payoff_curves{price_range[], unhedged[], covered_call[], protective_put[]}} |
| Auth required | no |
