## [Architect] Design

### 1. Feature Summary

This refactor eliminates duplicated HTTP, formatting, and initialization logic from the `dashboard/js/fno/` module tree by extracting `fno/api.js` init functions into a new `fno/app-init.js` module, rewriting `fno/api.js` as a thin re-export of `shared/api.js`, and wiring `fno/utils.js` to re-export the generic helpers from `shared/utils.js` while retaining fno-specific formatters locally. All fno/ modules that currently import `apiBase` from `fno/api.js` are updated to import it from `fno/api.js` (which itself re-exports it from `shared/api.js`), preserving the existing import surface for those consumers without touching their import paths.
